功能:

  • 可以对web下资源的访问
  • 可以自定义图标
  • mapper目录为私有目录
  • 提供了映射类(sws文件(简单的中文重定向),class文件(这个类似于servlet))
  • 支持重定向,转发
  • 支持热更新

使用映射类需要用到此项目下的jar包,包含shendi.web.server.sws.ShendiWebServer接口,继承此接口来实现(类似于servlet)

配置文件 main.properties 包含其他配置文件路径信息
配置文件 config.properties 包含服务器配置信息
图标文件 可以自行替换 默认在 项目的 web 文件夹下(替换的文件名必须一致)
配置文件 mapper.properties 包含所有映射的配置

所有映射的类需实现 shendi.web.server.sws.ShendiWebServer 接口,并且需要在mapper文件夹或其子文件夹下

不同于tomcat和其他web服务器,我这里支持文本形式(中文)进行简单的重定向(后期扩展)

需要使用不为class的文件来进行映射,文件后缀必须为 .sws 例如 Hello.sws
