# 玩法:
>车走直,吃一直线不隔子<br>
>马走日,马路前无子才能走<br>
>相飞田,田中无子才能飞<br>
>仕护主,斜上斜下不出格<br>
>兵往前,一格一格向前走,过河可以左右走<br>
>炮走直,炮隔一子可打子.<br>
	
# 人机算法
>智障算法,有棋则吃,无棋则随机行走<br>
